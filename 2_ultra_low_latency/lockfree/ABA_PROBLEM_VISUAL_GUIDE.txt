┌───────────────────────────────────────────────────────────────────────────┐
│ SOLUTION 1: TAGGED POINTERS (Version Counter)                            │
├───────────────────────────────────────────────────────────────────────────┤
│                                                                           │
│  Instead of:  std::atomic<Node*> head;                                   │
│  Use:         std::atomic<{Node*, uint64_t}> head;                       │
│                                  ↑                                        │
│                            Version tag                                    │
│                                                                           │
│  T0: Thread 1 reads {ptr=A, tag=1}                                       │
│      head = {A, 1} ───→ {B, 0} ───→ {C, 0}                               │
│                                                                           │
│  T1: Thread 2 pops A                                                      │
│      head = {B, 2} ───→ {C, 0}     (tag incremented!)                    │
│                                                                           │
│  T2: Thread 2 pops B                                                      │
│      head = {C, 3}                  (tag incremented!)                    │
│                                                                           │
│  T3: Thread 2 pushes A (reused)                                           │
│      head = {A, 4} ───→ {C, 3}     (tag incremented!)                    │
│                                                                           │
│  T4: Thread 1 resumes: CAS({A,1}, {B,2})                                 │
│      Expected: {A, 1}                                                     │
│      Actual:   {A, 4}                                                     │
│      ❌ FAILS! Tag mismatch (1 ≠ 4)                                       │
│      ✅ NO CORRUPTION! Thread 1 retries with new value                    │
│                                                                           │
│  Performance: +5-10ns overhead per operation                              │
└───────────────────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────────────────┐
│ SOLUTION 2: HAZARD POINTERS (Safe Memory Reclamation)                    │
├───────────────────────────────────────────────────────────────────────────┤
│                                                                           │
│  Idea: Threads "announce" which pointers they're using                    │
│                                                                           │
│  std::atomic<Node*> hazard_pointers[MAX_THREADS];                        │
│                                                                           │
│  Thread 1: hazard[0] = A  ← "I'm using A, don't delete it!"              │
│  Thread 2: hazard[1] = B  ← "I'm using B, don't delete it!"              │
│                                                                           │
│  Before deleting node:                                                    │
│    if (is_hazard(node)) retire_later(node);  // Defer deletion           │
│    else delete node;                         // Safe to delete           │
│                                                                           │
│  Performance: +50-100ns overhead (scanning hazard array)                  │
└───────────────────────────────────────────────────────────────────────────┘
